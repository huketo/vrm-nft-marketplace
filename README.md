## ğŸš€ Getting Started

1. Clone git repo

```sh
git clone https://github.com/huketo/vrm-nft-marketplace.git
```

2. Install package

```sh
yarn install
# or
npm install
```

3. Start Local Node

```sh
npx hardhat node
```

4. Deploy smartcontract

```sh
npx hardhat run scripts/deploy.ts --network localhost
```

5. Build front-end

```sh
cd front
yarn build
# or
npm run build
```

6. Deploy server

```sh
cd server
yarn deploy
# or
npm run deploy
```
